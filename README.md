# Elementor Magazine Blog Grid Widget

A custom Elementor widget that displays posts in a magazine-style grid layout.

## Features

- Magazine Grid Layout
- Category Selection
- Date Display
- Pagination
- Typography Controls
- Text Shadow Controls
- Overlay Styling
